Deployed the site to the existing Cloudflare Pages project `ai-consulting` using Wrangler CLI.

Commands/actions:
- Confirmed Wrangler login and located existing Pages project `ai-consulting`
- Created a clean temporary deploy directory containing only `index.html`, `script.js`, `styles.css`, and `public/`
- Ran:
  `wrangler pages deploy <temp-dir> --project-name ai-consulting --branch main --commit-hash $(git rev-parse HEAD) --commit-message "$(git log -1 --pretty=%s)" --commit-dirty=true`
- Verified deployment appeared in `wrangler pages deployment list`
- Verified HTTP 200 responses for both the deployment URL and `https://consulting.isolated.tech`
